# Temperature Monitoring System (C++)

## Overview
This project simulates a temperature monitoring system that triggers warnings based on input values.

## Features
- Accepts temperature input
- Displays warning if temperature exceeds threshold
- Simple control logic

## Concepts Demonstrated
- Conditional statements
- Sensor-like input simulation
- Real-world system modelling

## Purpose
This project reflects how embedded systems monitor environmental conditions and respond to critical values.
